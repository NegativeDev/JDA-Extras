# JDA-Extras
Extra utilities for JDA

# Feature List
## Commands
Automatic command manager with the addition of the features provided
* Abstract Command API
* Abstract SubCommand API
* Command aliases
* SubCommand aliases
* Command Cooldowns
* SubCommand Cooldowns

## Other Utilities
* Time Translator
* Decimal Formatter
